Project Chatty App
=====================

An app like a chatroom implemented React and Websocket.


### Getting Started

1. Fork this repository, then clone your fork of this repository.
2. Install dependencies using the `npm install` command.
3. Go to chatty-server folder and use the `npm run start` command, the server will be on <http://localhost:3001/>.
4. The app will be served at <http://localhost:3000/>.
5. Go to <http://localhost:3000/> in your browser.


### Dependencies

- babel-core
- babel-loader
- babel-preset-react
- babel-preset-es2015
- css-loader
- node-sass
- sass-loader
- sockjs-client
- style-loader
- webpack
- webpack-dev-server
- react
- react-dom
- express
- ws
- uuid
